FAQ & Tutoriels

❓ Comment ajouter une nouvelle Box ?

Suivez ce tutoriel détaillé étape par étape.

  1. Accédez à l’accueil de l’application BeetleBox.
  2. Cliquez sur la carte contenant une icône « + ».
  3. Scannez le QR Code présent sur la box.
  4. Les informations s'affichent automatiquement.
  5. Complétez ou ajustez les champs de texte.
  6. Cliquez sur « Ajouter l’appareil ».

📶 Connexion MQTT (Adafruit)

Configurez votre ESP32 avec le broker MQTT.

Consultez le guide officiel de Adafruit :

https://io.adafruit.com/

⚠️ Problèmes de connexion

Dépannez les erreurs fréquentes avec l’ESP32.

Vérifiez vos identifiants MQTT, les connexions Wi-Fi et consultez :

https://support.adafruit.com/
